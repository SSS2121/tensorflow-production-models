# TensorFlow Production Models

Este repositorio contiene experimentos y modelos desarrollados con TensorFlow y Keras, desde la lógica fundamental de regresión hasta la clasificación de datasets complejos.

## Proyectos Incluidos
- **IA Basica (`ia_basica.py`):** Un modelo de regresión lineal simple que aprende la relación matemática entre grados Celsius y Fahrenheit. Demuestra los fundamentos de `tf.keras.Sequential`, compilación y predicción.
- **Clasificación de Ropa (`clasificacion_de_ropa.py`):** Clasificador multiclase utilizando el dataset *Fashion MNIST*. Implementa preprocesamiento de imágenes (`normalizar`), capas densas (`Dense`), activación `ReLU` y `Softmax`.

## Tecnologías
- Python, TensorFlow, Keras, TensorFlow Datasets (tfds), NumPy, Matplotlib.

## Lo que demuestra este repositorio
- Capacidad para estructurar modelos de aprendizaje supervisado.
- Manejo de pipelines de datos y transformación de imágenes para entrenamiento.
- Evaluación de métricas de precisión y uso de optimizadores como `Adam`.
